# wordslab-images - Image generation at home

Local implementation of a high quality image generator.

This implementation is based on the following projects

- https://blackforestlabs.io/flux-1/
- https://github.com/huggingface/diffusers
- https://www.gradio.app/

## Install all the prerequisites

1. Install and start wordslab-notebooks on your machine

https://github.com/wordslab-org/wordslab-notebooks#wordslab-notebooks---ai-at-home

2. Open a terminal in Jupyterlab and execute the command

```bash
create-workspace-project https://github.com/wordslab-org/wordslab-images.git
```

3. In the file browser, navigate to the directory /wordslab-chat, then open the notebook

> 01_install_prerequisites.ipynb

4. Make sure the kernel 'wordslab-images' is selected, then execute all cells from the notebook

## Use Flux.1 schnell imagbe generator

1. Open the notebook:

> 02_start_and_use_flux1.ipynb

2. Make sure the kernel 'wordslab-images' is selected, then execute all cells from the notebook and follow the instructions

3. Watch these tutorials to learn how to use Flux.1

[How to Prompt FLUX. The BEST ways for prompting FLUX.1 SCHNELL and DEV including T5 and CLIP](https://www.youtube.com/watch?v=OSGavfgb5IA)

[The Most Comprehensive and Detailed Flux User Manual: Learn Everything About Flux in 30 Minutes](https://www.youtube.com/watch?v=vBaB_YmoZ-0)
